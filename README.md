# 🧪 Counter-Strike 1.6 (ReHLDS) — Pterodactyl Egg

**🇺🇸 English** | [🇷🇺 Русский](README.ru.md)

A powerful [Pterodactyl](https://pterodactyl.io) egg to deploy and manage a **Counter-Strike 1.6** server using **ReHLDS** and popular modules.

---

## 🚀 Features

- 🔧 Automated installation via SteamCMD and GitHub releases
- 🧩 Modular setup: `rehlds`, `regamedll`, `reapi`, `metamod-r`, `amxmodx`, `reunion`, `revoice`
- 💡 Full control via environment variables
- 🐧 Linux (x86) compatible
- 🔄 Auto-update support
- ✅ Fully plugin-compatible (AMXX, plugins, anti-cheats)

---

## 📦 Included Modules

| Module        | Description                                  |
|---------------|----------------------------------------------|
| **ReHLDS**    | Rewritten and optimized HLDS binary          |
| **ReGameDLL** | Custom game logic for CS 1.6                 |
| **ReAPI**     | Advanced API for modding                     |
| **MetaMod-R** | Module loader                                |
| **AMX Mod X** | Plugin scripting and admin control           |
| **ReUnion**   | Steam emulation (optional)                   |
| **ReVoice**   | Voice chat support (optional)                |

---

## 🛠 Installation

1. Go to the Pterodactyl Admin panel
2. Navigate to **Nests → Import Egg**
3. Upload `egg-counter-strike1-6-rehlds.json` or paste its raw URL
4. Assign the egg to your desired nest (e.g., “GoldSrc Games”)
